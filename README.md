# 熊浩·跟熊浩学沟通·30讲

> 学习笔记与实践应用

## 课程简介

沟通，是为了实现具体目标而进行一场有效对话。学会沟通，这个技能会贯通你所有的生命场景。

**讲师：** 熊浩·复旦大学副教授、谈判专家

## 学习目标

- ✅ 掌握沟通的基本原则（结构、性质、目的、风格）
- ✅ 学习谈判的理论与实战技巧
- ✅ 提升职场沟通能力（谈升职、跨部门协作等）

## 课程结构

1. **沟通原则** — 结构、性质、目的、风格四个奠基观念
2. **沟通主脉** — 哈佛大学原则谈判理论与流程（12讲）
3. **沟通拓展** — 说服、辩论、冲突管理
4. **沟通精炼框架** — 化零为整的实战纲领
